# Projet Web VISION

Application web actuellement en développement.

VISION est un projet web évolutif composé :

- D’une version statique fonctionnelle (HTML / CSS / JavaScript)
- D’une version dynamique en cours de développement (PHP + MySQL)

Le projet évolue progressivement vers une application web complète.

---

## 🚧 État du projet

✔ Version 1 : Site statique multi-pages (terminée)  
🚧 Version 2 : Intégration backend PHP + base de données MySQL (en cours)

---

Projet-Web-VISION/

## 💡 Fonctionnalités actuelles

- Site multi-pages en HTML/CSS
- Navigation fluide entre sections
- Présentation de contenu (cours, informations)
- Structure organisée des assets (CSS, JS, images)

---

## 🔄 Backend en cours de développement

La prochaine version inclut :

- Intégration PHP
- Connexion à une base de données MySQL
- Gestion via phpMyAdmin
- Dynamisation des contenus
- Interaction serveur (formulaires, données dynamiques)

L’objectif est de transformer le site statique en application web complète avec une architecture full-stack.

---

## 🌐 Déploiement prévu

Une fois le développement backend finalisé, le projet sera mis en ligne dans une phase de test (environnement de pré-production) afin de :

- Vérifier la stabilité de l’application
- Tester les fonctionnalités en conditions réelles
- Valider les interactions base de données / serveur
- Corriger les éventuels bugs avant mise en production

Dans cette optique, une solution d’hébergement gratuite supportant PHP et MySQL, telle qu’InfinityFree, est envisagée pour le déploiement initial.  
Ce choix permettra de tester l’application dans un environnement réel tout en maîtrisant les coûts avant une éventuelle migration vers une solution plus avancée.
